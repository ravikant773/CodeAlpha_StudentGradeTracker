# Student Grade Tracker

## Project Description
Student Grade Tracker is a Java-based console application that helps manage student records and calculate grade statistics.

The application allows users to add student details, view student reports, calculate average scores, find highest and lowest grades, and search for a specific student.

## Features

- Add student details (ID, Name, Grade)
- View all student records
- Calculate average score
- Find highest and lowest score
- Search student by name
- Grade category calculation (A, B, C, D, F)
- Grade validation (0-100 range)
- Menu-driven console interface

## Technologies Used

- Java
- ArrayList
- Scanner Class
